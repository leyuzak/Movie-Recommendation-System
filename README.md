## ğŸ¤– Model Overview
The recommendation engine is based on **Item-Based Collaborative Filtering** using **cosine similarity**.

### Key characteristics:
- no iterative training required
- similarity computed between movie vectors
- only **top-K most similar items** per movie are stored
- very small model artifact size (suitable for GitHub & Hugging Face)

### Recommendation logic:
User preferences are inferred from previously rated movies, and recommendations are generated by aggregating similarity-weighted scores from neighboring items.
